Desenvolvi uma calculadora utilizando o Delphi 13, com o objetivo de praticar lógica de programação, manipulação de eventos e construção de interfaces gráficas.

A aplicação permite ao usuário realizar as quatro operações matemáticas básicas:

Adição
Subtração
Multiplicação
Divisão

A interface foi construída utilizando componentes visuais como botões, campos de texto e rótulos, proporcionando uma interação simples e intuitiva. Cada botão numérico adiciona valores ao visor, enquanto os operadores definem a operação a ser realizada.

Durante o desenvolvimento, implementei:

Controle de entrada de dados em dois visores (primeiro e segundo número)
Identificação da operação selecionada
Cálculo do resultado ao pressionar o botão de igual
Limpeza dos dados para novos cálculos

Além disso, trabalhei conceitos importantes como:

Estruturas condicionais (if)
Conversão de tipos (String para Float)
Organização de código em eventos
Reutilização de lógica para evitar repetição

Esse projeto foi importante para reforçar minha base em programação e entender melhor como aplicações desktop funcionam na prática.
